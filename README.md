# tweetkiller
Gain score points by clicking on tweets. Fast. Burn 'em all!

## Instructions:
Pretty straight forward: clone, install dependencies, add keys & tokens, burn tweets🔥!

### Clone: 
Clone repository either by `https://github.com/Momciloo/tweetkiller.git` or by [downloading](https://github.com/Momciloo/tweetkiller/archive/master.zip) the source.
### Installation
From root folder, either by `npm install` or `yarn` ❤️
### Keys, tokens
Add Twitter security keys and access tokens to your *.env* file. To get them, [create new app](https://apps.twitter.com/) and generate keys.
